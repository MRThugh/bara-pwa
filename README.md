# 🌬️ بَرا (Bara) – اپلیکیشن هواشناسی پیشرفته

**بَرا (Bara)** یک اپلیکیشن وب موبایل-اول (Mobile-First) برای پیش‌بینی و نمایش وضعیت آب‌وهواست.  
این اپلیکیشن با طراحی مدرن، پشتیبانی از PWA، حالت تاریک/روشن، و استفاده از API رایگان [Open-Meteo](https://open-meteo.com/) ساخته شده است.  

---

## ✨ ویژگی‌ها
- 📱 **Mobile-First و ریسپانسیو** (سازگار با موبایل، تبلت و دسکتاپ)
- 🌍 **جستجوی شهر** + دکمه «موقعیت من» (GPS)
- 🌤️ نمایش آب‌وهوا:
  - دمای لحظه‌ای  
  - وضعیت آسمان  
  - رطوبت  
  - سرعت باد  
- ⏰ **پیش‌بینی ۲۴ ساعته** (اسکرول افقی)  
- 📅 **پیش‌بینی ۷ روز آینده** (Grid زیبا)  
- 🌗 **تم روشن/تاریک** (با ذخیره‌سازی در LocalStorage)  
- ⚡ **PWA کامل**:
  - نصب روی گوشی  
  - Splash Screen اختصاصی  
  - پشتیبانی از حالت آفلاین با Service Worker  
- 🌀 طراحی مینیمال و مدرن با TailwindCSS
